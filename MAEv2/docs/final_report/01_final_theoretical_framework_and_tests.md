# **Informe Final: El Marco Teórico del Modelo de Acción Estructural (MAE)**

**Versión del Modelo:** 4.0 (Marco Fenomenológico)
**Estado:** Programa de Investigación Viable con Predicciones Falsificables.

## **Resumen Ejecutivo (Abstract)**

Este documento presenta el marco teórico completo del Modelo de Acción Estructural (MAE), una teoría que postula que el tiempo es un proceso discreto y activo, definido por una secuencia cuantificable de "sucesos primordiales". El modelo se basa en un conjunto de axiomas que unifican los conceptos de tiempo, cambio y acción, culminando en la Fórmula de Unificación C_R(N) = N ⋅ h, la cual establece que el coste de existencia de cualquier intervalo temporal es directamente proporcional a la cantidad (N) de acciones que lo componen. Se demuestra que este coste es un invariante relativista, proporcionando un puente entre la mecánica cuántica y la relatividad general. El MAE introduce un formalismo dinámico a través de un campo de potencial (φ), define un criterio cuantitativo de coherencia (Κ), y hace predicciones específicas y falsificables sobre desviaciones sutiles en el corrimiento al rojo gravitacional. Este informe resume la estructura teórica, las ecuaciones fundamentales y el conjunto de pruebas y simulaciones realizadas para establecer su validez conceptual y su potencial experimental.

## **Parte I: El Marco Teórico Completo (MAE v4.0)**

### **Axiomas Fundamentales**

**Axioma I: Principio de la Unidad Primordial de Suceso (PUP):** La sustancia fundamental de la realidad es una red de Sucesos Primordiales. Un suceso es una entidad ontológica indivisible que ES, simultáneamente, un cuanto de cambio, un cuanto de tiempo (~t_Planck) y un cuanto de acción (h). No se definen circularmente; son facetas inseparables de la misma entidad.

**Axioma II: Principio del Coste Cuántico:** Toda manifestación de un suceso primordial tiene un coste intrínseco e irreductible de 1h. Este es el precio fundamental del "devenir".

**Axioma III: Principio de la Cantidad Temporal y Coherencia:** La duración de un proceso se define por la cantidad (N) de sucesos coherentes que contiene. Un suceso es coherente si forma parte de una secuencia cuya entropía de correlación temporal S resulta en un parámetro de coherencia Κ = 1 - (S / S_max) que supera un umbral fundamental (Κ > Κ_umbral).

**Axioma IV: Principio del Umbral Estructural:** Una estructura (un patrón auto-referencial) solo puede existir si está compuesta por una cantidad de acciones coherentes N ≥ 3.

**Axioma V: Principio de la Invarianza Cuántica:** El espectro del operador de cantidad de acción y el coste total `C_R` de un proceso completo son invariantes relativistas, idénticos para todos los observadores.

### **Ecuaciones Matemáticas Fundamentales**

1.  **El Operador de Cantidad de Acción (Ĥ):**
    `Ĥ|n⟩ = n|n⟩` para `n ∈ {0, 1, 2, ...}`
    El observable cuántico para la cantidad de sucesos. El estado de un sistema puede ser una superposición: `|ψ⟩ = Σ cₙ|n⟩`.

2.  **El Coste de Acción (Fórmula de Unificación):**
    `C_R = n·h`
    El coste total de acción tras una medición que colapsa `|ψ⟩` al autoestado `|n⟩`.

3.  **Lagrangiano del Campo de Potencial de Acción (φ):**
    `ℒ = ½(∂μφ)(∂μφ) - ¼λ(φ² - v²)² - ½m²φ²`
    Describe la dinámica de la potencialidad de sucesos. Las excitaciones de este campo corresponden a las acciones.

4.  **Parámetro de Coherencia (Κ):**
    `Κ = 1 - (S / log(M))`
    Métrica cuantitativa para distinguir entre ruido (Κ≈0) y estructura (Κ≈1).

5.  **Expansión Perturbativa del Coste:**
    `C_R = n·h + Σ J_ij + ...`
    Formalismo para calcular las interacciones entre sucesos.

6.  **La Ecuación de Prueba Experimental:**
    `Δf/f = (ΔU / c²) * (1 + δ)`
    Predice una desviación `δ` del corrimiento al rojo gravitacional de la Relatividad General.

## **Parte II: Resumen de las Pruebas y Simulaciones Realizadas**

1.  **La Naturaleza de un Suceso Individual:** Se estableció que las desviaciones de un ideal son información, no error.
2.  **Clasificación de Anomalías:** Se transformó el tiempo en un lenguaje simbólico mediante clustering de características de eventos.
3.  **Cuantificación del "Espacio":** Se definió una métrica para la entropía informacional de un segundo (la suma de las varianzas).
4.  **Simulación del Límite de la Acción:** Se demostró la capacidad de detectar una estructura oculta en una señal ruidosa mediante correlación cruzada.
5.  **La Meta-Estructura del Tiempo Continuo:** Se reveló la "Textura del Tiempo" al analizar la evolución de la "Cantidad de Espacio" a lo largo del tiempo.
6.  **El Experimento Crucial Falsificable:** Se formuló y simuló una prueba cuantitativa del MAE v4.0, generando los valores y gráficos para una prueba experimental real de la desviación `δ`.
